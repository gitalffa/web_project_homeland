## 📋 Descripción

**De Patria a Patria** es una página editorial que presenta historias y fotografías de los lugares de origen de los profesionales del equipo TripleTen. El proyecto pone énfasis en tipografía editorial, layout responsivo con Flexbox y CSS Grid, y estructura BEM anidada con archivos separados por bloque.

Cada lugar presentado incluye fotografía, descripción histórica y cultural, créditos del artista y un botón para "comprar la obra como NFT".

## ✨ Secciones del proyecto

- **Header** — Logo de TripleTen, título y subtítulo del viaje
- **Intro editorial** — Cita de Joseph Beuys y texto de bienvenida de la galería
- **Photo Grid** — Galería de 8 fotografías de los distintos lugares
- **Places** — Tarjetas editoriales de cada ciudad con imagen, descripción y botón NFT
- **Footer** — Créditos del autor

## 🛠 Tecnologías

| Tecnología              | Uso                                                                    |
| ----------------------- | ---------------------------------------------------------------------- |
| HTML5 semántico         | Estructura editorial con `<header>`, `<main>`, `<section>`, `<footer>` |
| CSS3                    | Estilos, tipografía y layout responsivo                                |
| Flexbox y CSS Grid      | Galería de fotos y layout de tarjetas                                  |
| Metodología BEM anidada | Clases organizadas por bloque en carpeta `blocks/`                     |

## 📐 Conceptos aplicados

- **BEM anidado** — Cada bloque tiene su propia carpeta y archivo CSS dentro de `blocks/`
- **CSS Grid** — Layout de la galería de fotografías
- **Flexbox** — Tarjetas de lugares y estructura general
- **Tipografía editorial** — Jerarquía visual con diferentes pesos y tamaños
- **HTML5 semántico** — Uso correcto de elementos de sección y estructura
- **PWA básico** — Web App Manifest y set completo de favicons
